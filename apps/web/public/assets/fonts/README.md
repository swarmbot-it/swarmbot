# Fonts

Self-hosted **Plus Jakarta Sans** (400–800) and **JetBrains Mono** (400–600), latin + latin-ext subsets (WOFF2).

Both families are under the [SIL Open Font License 1.1](https://openfontlicense.org/).

Source: [Fontsource](https://fontsource.org/) (Google Fonts). To refresh files, install `@fontsource/plus-jakarta-sans` and `@fontsource/jetbrains-mono`, then copy `*-latin-*-normal.woff2` and `*-latin-ext-*-normal.woff2` from each package’s `files/` directory.
